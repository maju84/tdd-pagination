# tdd-pagination 7
a code kata about pagination using tdd

```
there are just five pages, we are on page 2
1 (2) 3 4 5


there are seven pages, we are on page 5
1 2 3 4 (5) 6 7

there are 103 pages, we are on page 42
1 ... 41 (42) 43 ... 103

there are 103 pages, we are on page 3
1 2 (3) 4 5 ... 103

there are 103 pages, we are on page 102:
1 ... 99 100 101 (102) 103

-> if possible show up to seven pages
-> always show first and last page on position 1 and 7
-> show predecessor and successor of current page 


```
